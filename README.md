# 🎬 CineManuuu!

A modern and responsive Movie Search Web Application built using HTML, CSS, and JavaScript. Discover trending movies, search for your favorite films, and view detailed movie information using the TMDB API.

## ✨ Features

- 🔍 Search movies instantly
- 🎬 Browse trending movies
- ⭐ View movie ratings
- 📅 Check release dates
- ⏱️ View movie runtime
- 📖 Read movie overviews
- 📱 Fully responsive design
- 🎨 Modern and clean user interface
- ⚡ Fast loading experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TMDB API

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/cinemanuuu.git
```

### 2. Open the Project

Open the project folder and launch `index.html` in your browser.

### 3. Configure TMDB API

Open `script.js` and replace the API key with your own TMDB API key:

```javascript
const API_KEY = "YOUR_API_KEY_HERE";
```

## 🎯 How It Works

1. The application loads trending movies on startup.
2. Users can search for movies using the search bar.
3. Movie cards display posters, titles, and ratings.
4. Clicking the "Details" button opens a popup with additional movie information.
